# 🎬 Sistema de Cinema
Sistema web desenvolvido para gerenciamento de um cinema, permitindo o cadastro de filmes, salas, sessões e a venda de ingressos.

## 🚀 Funcionalidades

- Cadastro de filmes
- Cadastro de salas
- Cadastro de sessões
- Venda de ingressos
- Listagem de sessões

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- LocalStorage

## ▶️ Como executar o projeto

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Utilize o sistema normalmente

## 📁 Estrutura do projeto

- index.html
- cadastro-filmes.html
- cadastro-salas.html
- cadastro-sessoes.html
- venda-ingressos.html
- sessoes.html
- js/script.js
